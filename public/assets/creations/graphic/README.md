# Graphic Design Assets

This directory contains graphic design work including posters, brochures, and branding materials.

## Required Assets

### Current Placeholders Needed:
1. **poster-placeholder.jpg** (400x225px)
   - Neon dreams poster series
   - Can be temporary stock image or generated placeholder

2. **brochure-placeholder.jpg** (400x225px)
   - Modern tech brochure design
   - Can be temporary stock image or generated placeholder

## Adding New Graphic Design Assets

1. Export your design at 400x225px (16:9 aspect ratio) for thumbnails
2. Use descriptive kebab-case filenames: `my-design-name.jpg`
3. Update `src/data/creations.ts` with the new entry in the `GRAPHIC` array

## Recommended Formats
- **JPG**: For most designs (good compression)
- **PNG**: For designs with transparency or text-heavy layouts
- **PDF**: For print-ready versions (not for web thumbnails)

## Optimization
- Compress images before adding (use tinypng.com)
- Target < 100KB per thumbnail
- Save print assets separately if needed
